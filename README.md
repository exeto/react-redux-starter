> React Redux Starter

**THIS REPOSITORY IS NO LONGER MAINTAINED**

## Install

```bash
$ git clone https://github.com/exeto/react-redux-starter.git
$ cd react-redux-starter && npm i
```

## Usage

```bash
# Development
$ npm run dev

# Production
$ npm run build
$ npm run start
```
